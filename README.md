# Weather App

A visually appealing weather application that provides real-time weather updates, hourly forecasts, and a 5-day weather forecast. The app dynamically changes its background based on the time of the day.

## Features

- Real-Time Weather Data: Fetches live weather details, including temperature, humidity, and weather conditions.
- Hourly Forecast: Displays upcoming weather changes in a detailed hourly format.
- 5-Day Forecast: Provides an extended forecast to help plan ahead.
- Dynamic Backgrounds: The background updates based on the time of day (day/night).
- Responsive Design: Optimized for desktop, tablet, and mobile devices.

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/talibkhan172001/weather-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd weather-app
   ```

3. Open `index.html` in a web browser or use a local server.

## Usage

- Enter the desired city name in the search bar.
- View current weather details including temperature, humidity, and weather conditions.
- Scroll through the hourly forecast for upcoming weather trends.
- Check the 5-day forecast for future weather predictions.

## File Structure

- `index.html` - Main structure of the application.
- `style.css` - Styling for layout and responsiveness.
- `main.js` - JavaScript logic for API calls and UI updates.
- `day-video.mp4` & `night-video.mp4` - Background videos for different times of the day.

## API Key

To run the project, you need an API key from [OpenWeatherMap](https://openweathermap.org/api):
- Replace `YOUR_API_KEY` in `main.js` with your API key.
